# Light-Pulse

Light-Pulse is an interactive playground where visuals and sound react to your cursor.  
Move around inside the pad box to paint it with colors and shape the tones in real-time.  

---

## What you can do
- Watch the pad box light up with shifting colors as you move your mouse.
- Play with sound that changes pitch and volume depending on where the cursor goes.
- Toggle the sound **On / Off** with a single button (colors always stay active).

---
## Getting started :
1. Clone the workspace repository: `https://github.com/yanas-logs/archive-workspace.git`
2. Move into the project directory: `cd archive-workspace/apps/light-pulse`
3. Open the `index.html` file in a web browser to see the project in action.
4. Move your cursor over the box area to see the background color change.
